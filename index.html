HTML Tags Documentation

This page provides a brief overview of common HTML tags with examples.

1. Headings

HTML provides six levels of headings:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

2. Paragraph and Text Formatting

HTML provides tags for text styling:

<p>This is a paragraph.</p>
<b>Bold Text</b>, <i>Italic Text</i>, <u>Underlined Text</u>

3. Lists

Unordered List:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Ordered List:

<ol>
    <li>First</li>
    <li>Second</li>
    <li>Third</li>
</ol>

4. Links

Creating hyperlinks:

<a href="https://www.example.com">Visit Example</a>

5. Images

Adding an image:

<img src="image.jpg" alt="Description" width="300">

6. Tables

Creating a table:

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>

Rendered Table:

Name Age
John 25

7. Forms

Creating a simple form:

<form action="#" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name"><br><br>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br><br>

    <label for="message">Message:</label><br>
    <textarea id="message" name="message"></textarea><br><br>

    <input type="submit" value="Submit">
</form>

Rendered Form:








8. Buttons

Creating a button:

<button>Click Me</button>

9. Div and Span

Block-level and inline elements:

<div style="background: lightgray; padding: 10px;">
    This is a div (block-level).
</div>
<span style="color: red;">This is a span (inline).</span>

10. Audio and Video

Embedding media:

<audio controls>
    <source src="audio.mp3" type="audio/mp3">
</audio>

<video controls width="300">
    <source src="video.mp4" type="video/mp4">
</video>
